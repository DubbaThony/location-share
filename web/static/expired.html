Session Expired

← Back to main

This share session either expired or never existed in the first place. The server doesn't keep records of past sessions, so we genuinely can't tell you which.

Usually this means one of:

  • The sender ended the share (toggled it off, or the share-timer ran out).
  • The server was restarted at some point after the link reached you.
  • The link got mangled in transit (truncated, copy-paste split, the access key in ?id=... partially eaten).

If you trust the source of the link, the simplest fix is to ask them to start a new share and send you a fresh URL. Each share gets a brand-new ephemeral access key and a brand-new inner encryption key - they're deliberately single-use.
